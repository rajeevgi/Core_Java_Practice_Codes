## 🧠 How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/java-revision.git
   cd java-revision
   ```

2. Compile the Java files:
   ```bash
   javac ControlStatements/Practice.java
   ```

3. Run the program:
   ```bash
   java ControlStatements.Practice
   ```
